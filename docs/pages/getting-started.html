Getting Started

All requests to the API must be sent to http://amaportalapi.besaba.com/api/


Currently, GET requests sent to the API do not need any security tokens attached. On the other hand, POST requests would require an access_token attached in every requests to ensure security and reliability. Your app may acquire this token upon your client's login through Oauth2 authentication. You may save the access_token in a PHP Session for later use.


All reponse from the API will be in JSON format.


On the left side is a menu containing the list of currently availabe API on this portal. Select an API to find out more information in using and accessing each one of them.
