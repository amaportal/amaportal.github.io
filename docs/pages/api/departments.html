API > Getting Started

The departments API is used to access the list of departments exisiting in AMA Davao.


How To Use
Select from the table below the API Link that you need to use for your application. Just add the API Link to the current API address provided here. ie http://linktoapi.com/api/departments.

API Link Description
GET Requests
departments Get the list of all existing departments.
departments/results/count Get the list of departments with pagination. Change the count value to the number of results you need per page.
departments/id Get the department with an ID equal to the id value on the API link.
departments/id/programs Get the list of Programs offered under the selected Department.
departments/id/programs/results/count Paginate the number of Programs under the selected Department. Limit the results per page to the count value.
